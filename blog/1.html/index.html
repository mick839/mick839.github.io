1、博客上传到github，并且绑定域名：mick.wtf

2、开启自动部署，步骤如下：

（1）本地更新文件

在本地修改内容，如增加、修改博客，或修改前端代码

（2）push到github

所需代码如下：

cd d:/hugo-blog
d:
git add .
git commit -m "更新了博客"
git push -u origin master会自动action并发布到静态仓库

完成，等待自动更新即可~
